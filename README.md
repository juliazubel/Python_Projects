# Data Science & Data Engineering Projects

This repository contains a collection of my academic and personal projects related to **data analysis, machine learning, financial data processing, and SQL database design**.

The goal of these projects is to demonstrate practical experience with **Python, SQL, data pipelines, machine learning models, and data analysis workflows**.

Most projects were developed as part of coursework or self-directed learning and focus on solving real-world analytical problems using modern data science tools.

---

## Repository Structure

The repository includes multiple projects covering different areas of data science and analytics:

### Data Analysis & Data Engineering

* **Financial Data Analysis Pipeline** – automated pipeline for collecting, processing, and analyzing financial datasets.

### Machine Learning

* **Machine Learning – Yamaha Stock Price Prediction with LSTM** – deep learning model for time series forecasting of stock prices using an LSTM neural network.

### Databases & SQL

* **SQL Cinema Database** – relational database system designed for cinema management, including ticket sales, screenings, customers, employees, and revenue tracking.

Additional projects explore topics such as data visualization, exploratory data analysis, and predictive modeling.
